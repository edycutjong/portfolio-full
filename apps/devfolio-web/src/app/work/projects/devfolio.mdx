## Overview

DevFolio is a modern developer portfolio website featuring an integrated AI chatbot that can answer questions about my skills, projects, and experience 24/7. Built with Next.js 15 and deployed on Vercel for optimal performance.

## Key Features

- **AI Chatbot**: GPT-4o-mini powered assistant with context about my portfolio
- **Modern Design**: Clean, responsive design with dark mode support
- **Project Showcase**: Interactive project cards with filtering by category
- **Fast Performance**: Static generation with incremental regeneration

## Technologies Used

- **Next.js 15**: React framework with App Router
- **TypeScript**: Type-safe development
- **Tailwind CSS**: Utility-first styling
- **Hono**: Lightweight API routes
- **Vercel**: Edge deployment platform

## AI Integration

The chatbot uses:
- Custom system prompts with portfolio context
- Streaming responses for better UX
- Rate limiting and abuse prevention
- Conversation memory within sessions

## Outcome

This portfolio demonstrates full-stack capabilities, combining modern frontend development with AI integration for a unique user experience.
